# Gilded Rose Refactoring approche
* Create gold branch contains the orginal code only.  
* Write test cases that covers all possible scenarios before refactoring.
* Remove duplicated logic from updateQuantity.
* Use Item wrapper. No need to change main item class.
* Add class per each product.
